# Projects

Store your project notes here. 

## Getting Started
Use the project template from `templates/project-template.md` to get started.

## Naming Convention
One file per project, named with lowercase and hyphens:
- `my-awesome-project.md`
- `client-website-redesign.md`
- `api-authentication-system.md`

## Example Usage
1. Copy `../templates/project-template.md`
2. Save as `projects/your-project-name.md`
3. Fill in the sections as your project evolves
