# Learning

Store your learning notes here.

## Getting Started
Use the learning template from `templates/learning-template.md` for courses, tutorials, and topics you're studying.

## Naming Convention
One file per topic/course:
- `react-hooks-tutorial.md`
- `machine-learning-basics.md`
- `docker-fundamentals.md`

## Tips
- Take notes as you learn
- Add code examples
- Link to related topics
- Track your progress
- Create practice projects
